Trigger

刺激 Trigger

  • python环境，使用serial module发送mh##格式marker的对应关系

    marker字符 Trigger
    mh00 R3
    mh10 R3 + S1
    mh20 R3 + S2
    mh30 R3 + S3
    mh40 R3 + S4
    mh50 R3 + S5
    mh60 R3 + S6
    mh70 R3 + S7
    mh80 R3 + S8
    mh90 R3 + S9

  • matlab环境，在WriteToDevice函数中，argument (‘mh%c%c’,###,0)，中间数字对应的marker可以用这个数学关系表示

    mhN=RN∣16+SN mod 16 \text{mh}N = R_{N\mid16} + S_{N \bmod 16} mhN=RN∣16​+SNmod16​

    举例，发送mh199，则199整除16为12余7，则对应的trigger显示为R12+S7.
